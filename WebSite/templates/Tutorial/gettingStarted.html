Getting started

This Tutorial shows you how to setup your system for the following tutorials.
All Tutorials assumes that you are familiar with CORBA, .NET Remoting and RMI/IIOP.

Requirements

To do the tutorials you need

  • .NET Framework
  • JDK 1.4
  • IIOP.NET Channel

Building the channel

If you are using the source distribution of IIOP.NET, you need to build the channel first.
For this task a Makefile is provided. Simple run nmake in the IIOP.NET directory.
