Compatibility

  1. Does IIOP.NET work with Microsoft's .NET 1.1?

Does IIOP.NET work with Microsoft's .NET 1.1?

Yes, it does. This is in fact the platform it was developed on.

Does IIOP.NET work with mono?

Currently (mono 0.24) not. This is due to the incomplete remoting implementation of mono. We hope this will change in the future.
